# 🚀 .gitignore 功能快速指南

## 一分鐘了解新功能

ReplaceText 現在會**自動排除** `.gitignore` 中定義的檔案, 讓您的專案更安全!

## 📖 使用方式

### 基本使用 (自動啟用)

```bash
# 就像以前一樣使用,程式會自動套用 .gitignore 規則
replacetext /T /path/to/project
```

### 看看哪些檔案被忽略

```bash
# 使用 /V 參數查看詳細資訊
replacetext /T /V /path/to/project
```
